# ManagerTG

Telegram бот для автоматического управления участниками групп и чатов.

## Описание

ManagerTG автоматически отслеживает новых участников в Telegram группах, сохраняет их данные в локальную базу и позволяет администраторам очищать чат от удаленных аккаунтов.

## Основные возможности

- Автоматическое добавление новых участников в базу данных
- Синхронизация существующих участников чата при запуске
- Удаление deleted accounts из группы по команде
- Просмотр статистики участников
- Работа с правами администратора

## Команды

- `/cleanusers` - удалить всех deleted accounts из чата
- `/dbusers` - показать количество пользователей в базе данных для текущего чата

## Стек технологий

- Python 3.8+
- Telethon - асинхронная библиотека для Telegram API
- aiosqlite - асинхронная работа с SQLite базой данных
- asyncio - асинхронное выполнение

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Отредактируйте `config.py` и добавьте свои данные:

```python
API_ID = "your_api_id"
API_HASH = "your_api_hash"
BOT_TOKEN = "your_bot_token"
AUTO_SYNC_CHAT_ID = -1001234567890
ADMIN_USER_ID = 123456789
```

Получить API_ID и API_HASH можно на https://my.telegram.org

## Запуск

```bash
python run.py
```

## Требования

- Python 3.8 или выше
- Telegram Bot Token
- API credentials от Telegram
- Права администратора в целевой группе
